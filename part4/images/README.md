Place your custom card images in this folder.

Recommended filenames:

- `lake-house.jpg`
- `forest-cabin.jpg`
- `city-stay.jpg`

Recommended format:

- horizontal images
- at least 1200px wide
- `.jpg`, `.jpeg`, `.png`, or `.webp`

If a custom image is missing, the front-end automatically falls back to the local SVG artwork.
